Jedine JavaScript, jedine MeteorJS

Meteor je pomerne mladá, ale zato mocná technológia. Meteor nie je iba framework, je to platforma postavená na Node.js. Potrebuješ vedieť HTML, JavaScript, MongoDB, trochu CSS a môžeš začať vyvíjať real-time aplikácie. Celú aplikáciu píšeš v JavaScripte. Back-end jazyky/frameworky ako PHP, či Rails nepotrebuješ. Chceš vedieť viac? Hlasuj a prídi na prednášku!

Komu je přednáška určena?

Všetkým, ktorí majú aspoň základnú znalosť JavaScriptu a HTML a chcú sa dozvedieť viac o Meteore. Môžu prísť aj JavaScript hateri :)

Přednáška se nedostala do programu.
