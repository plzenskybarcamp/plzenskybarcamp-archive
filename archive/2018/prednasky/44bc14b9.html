Řekněte si o API!

Existence API neboli programátorského rozhraní webových aplikací není žádné tajemství, přesto jej většina českých webových aplikací stále nemá. API umožňuje automatizaci i doplnění funkčnosti, která vám bude šetřit čas a peníze. Pozor, díky chytrým nástrojům jako Zapier, Integromat či Dativery můžete API využívat snadno i bez znalosti programování.

Od vysvětlení podstaty API se přes jeho vhodnou podobu dostaneme především k tomu, jak API mohou využívat neprogramátoři.

Komu je přednáška určena?

Všem, kteří používají webové aplikace, ale chtěli by je využívate lépe i bez programování.

Přednáška se nedostala do programu.
